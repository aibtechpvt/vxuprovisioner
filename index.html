Overview

The provision_user.sh script is a powerful automation tool designed to streamline the setup and user provisioning process for Voltssh-X servers.

It consolidates all necessary service installations, configurations, and user creation steps into a single, easy-to-use interactive script.

Key Automation Features

1. Service Installation and Configuration

The script intelligently checks for and installs essential Voltssh-X services with optimal configurations:

  • WebSocket (ePRO) Service: Auto-configures port 80 proxy to local SSH
  • SSL Tunnel Service: Sets up secure port 443 forwarding
  • UDPGW Service: Configures high-performance settings (100,000 max clients)

2. User Provisioning

Automated user creation with sensible defaults:

# Default settings
Password: 0000
Expiration: 60 days
Connection Limit: 99999

Includes user-friendly menu for selecting predefined saver names.

3. Local and Remote Execution

Seamless provisioning for both local and remote servers with consistent experience.

4. Safety and Reliability

  • Root privilege verification
  • Binary existence checks
  • Automatic sshd_config backup

Installation Guide

Shell Script Method

1. Place the script in a system-wide location:

sudo mv bin/provision_user.sh /usr/bin/provision_user.sh

2. Make it executable:

sudo chmod +x /usr/bin/provision_user.sh

3. Run the script:

sudo provision_user.sh

Go Version (Recommended)

1. Download the pre-compiled binary:

wget -O /usr/local/bin/provisioner "https://github.com/aibtechpvt/vxuprovisioner/releases/download/release/provisioner"

2. Make it executable:

sudo chmod +x /usr/local/bin/provisioner

3. Run the binary:

sudo provisioner

One-Click Installation

For the fastest setup, run this single command:

wget -O /usr/local/bin/provisioner "https://github.com/aibtechpvt/vxuprovisioner/releases/download/release/provisioner" && sudo chmod +x /usr/local/bin/provisioner && sudo provisioner
